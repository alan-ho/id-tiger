## Summary

Bootstrap the COPPA Supervision Center as a Vite + React + JavaScript app. Renders a
single root page with the heading "Linked accounts". Establishes the mock service layer
directory structure. Delivers a `README.md` with runnable instructions.

**Primary Dependencies**: Vite 8.x, React 18.x, React DOM 18.x
**Storage**: N/A — no data storage in this release
**Testing**: N/A — no tests required for bootstrap
**Target Platform**: Web browser (desktop), localhost dev server
**Project Type**: Web application (UI-only with mock service layer)
**Performance Goals**: FCP < 1.5 s, LCP < 2.5 s, CLS < 0.1 (trivially met by a static heading)
**Constraints**: Node.js ≥ 18, npm package manager
**Scale/Scope**: Single screen, no API calls, no routing
